# battlesnake_crystal

Battlesnake 2018 written in Crystal because why not.


## Installation

Install Crystal off the website or just use:

```
brew update
brew install crystal-lang
```

Then to install deps run:

`crystal deps`


## Usage

To run the server, run:

`crystal server.rb`

## Issues
